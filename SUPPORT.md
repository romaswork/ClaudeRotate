## Frequently asked

**The app doesn't write to my settings file.**
Make sure you selected the target file in the "Settings" tab via the "Choose…" button. Because the app is sandboxed, it can only access the file you pick manually. If you upgraded from an older version, re-select the file once.

**My proxy test fails.**
A proxy test requires a real, reachable proxy. Error 407 means authentication failed (check username/password); a transport error means the proxy is unreachable.

**Where is my data stored?**
Locally on your Mac, in `~/Library/Application Support/ClaudeRotate/`. Nothing is sent to any server.

## Частые вопросы

**Приложение не пишет в файл настроек.**
Убедитесь, что выбрали целевой файл на вкладке «Настройки» кнопкой «Выбрать…». Из-за песочницы приложение имеет доступ только к файлу, который вы выбрали вручную. После обновления со старой версии выберите файл заново.

**Проверка прокси не проходит.**
Для проверки нужен реальный доступный прокси. Ошибка 407 — неверная авторизация (проверьте логин/пароль); транспортная ошибка — прокси недоступен.

**Где хранятся мои данные?**
Локально на вашем Mac, в `~/Library/Application Support/ClaudeRotate/`. Ничего не отправляется на серверы.
